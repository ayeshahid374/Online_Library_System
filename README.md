# 📚 Online Library Management System

A full-stack web application designed to manage and access digital books with dedicated user and admin functionalities. Built with React, Node.js, and SQL Server, it offers a responsive UI and efficient backend operations.

## ⚙️ Features

### For Users
1. Search and check book availability  
2. Bookmark and download ebooks  
3. View fines and activity  

### For Admins
1. Upload, edit, and delete books  
2. Manage users and monitor activity  
3. View top-performing books and insights  

## 🚀 How It Works
- Users and admins log in through a role-based system  
- Users interact with books (search, bookmark, download)  
- Admins manage content and track system data  
- Backend handles requests and database operations in real time  

## 🛠️ Tech Stack
- **Frontend:** React.js, HTML5, CSS3  
- **Backend:** Node.js, Express.js  
- **Database:** Microsoft SQL Server  
- **Version Control:** Git & GitHub
